Toast / inline notification

The single transient-feedback component, formalized so the "it worked" and light "it failed" messages already implied across the app (invite sent, saved, copied, and their failures) are consistent and accessible. A short message with an icon by shape (a solid check for success, a dashed "!" for error, a neutral i for info, never color alone), an optional action (Undo, Retry), a manual close, and an auto-dismiss timer, placed in a desktop corner stack or a mobile top or bottom slot, with aria-live and no focus trap. It is cleanly separated from the persistent inline panels (3.3, 10.6) and the modal dialog (0.5). No SEO impact. Source of truth: ia/docs/pages/toast.md.

Variants (meaning by shape, not color)

No color and no brand: structure only, color arrives at Concept (stage 06). Success is a solid check (the 10.3 mark), an error is a dashed "!" (the 2.3 mark), info is a neutral i; every toast pairs the mark with text, so the meaning never depends on color.

The feedback taxonomy

The reason to formalize the pattern: three feedback surfaces exist and must not be confused.

Accessibility and behavior

Accessibility and behavior

No SEO impact: a toast is transient UI, not content and not a URL. What matters is that it is announced and never traps the user.

  • aria-live: success and info use aria-live="polite" (announced without interrupting); errors use aria-live="assertive" or role="alert" so they are announced promptly
  • No focus trap: a toast never steals focus or blocks the page; if it carries an action (Undo, Retry), that control is keyboard reachable while the toast is present
  • Dismissable: the close control is always present; auto-dismiss is paused on hover or focus so the message can be read and acted on; errors persist longer or until dismissed
  • Reduced motion: the enter and exit is a simple fade when prefers-reduced-motion is set, no slide
  • State by shape and text: the success, info, and error marks differ by shape and fill and are paired with text, never distinguished by color alone
